hexo+github建立个人博客

发表于 2017-05-07 | 阅读次数

环境： windows10 64bits
所需软件：
　　1. Node.js　基于Chrome V8引擎的JavaScript运行环境；
　　2. Git-for-windows　windows平台下git 运行环境；
　　3. MarkdownPad　博客撰写工具

　　开始之前先去 github 注册一个账户，并新建一个与自己用户名同名的仓库，最后生成的仓库地址：https://github.com/username/username.github.io 注：username替换为自己的用户名
